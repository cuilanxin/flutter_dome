# flutter_dome

第一次使用 flutter 编写移动端项目

# 声明

- 本人并没有 dart 语言基础
- 本人使用的一直是是 js 偶尔使用 ts

# 安装依赖

```bash
# 安装所有依赖
$ flutter pub get
# 安装 english_words 依赖
$ flutter pub add english_words
# 跑项目 本人 mac 打开 Simulator
$ flutter run
```
